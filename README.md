# Movie-Explorer
A modern web app to explore movies with real-time data, built using JavaScript and public APIs.

## Overview
Movie Explorer is a modern, responsive web application designed to help users search and explore movies effortlessly using real-time data from a public API. The application focuses on delivering a clean user experience while demonstrating strong fundamentals of JavaScript, API integration, and dynamic UI rendering.

## Key Highlights
🔍 Real-time movie search experience
⚡ Fast and dynamic UI updates using JavaScript
🎯 Smart filtering and sorting using array higher-order functions
📱 Fully responsive design across all devices
🎨 Clean and modern user interface

## Features
### Smart Search
* Search movies instantly by title
* Optimized input handling (can include debouncing)

### Advanced Filtering
* Filter movies by:
  * Type (Movie / Series / Episode)
  * Release Year

### Sorting Options
* Alphabetical (A → Z / Z → A)
* Release Year (Newest → Oldest)

### Interactive Elements
* Add/remove favorites
* View detailed movie information

### User Experience
* Responsive design (mobile-first approach)
* Optional Dark Mode / Light Mode toggle

## API Integration
OMDb API (Open Movie Database) - https://www.omdbapi.com/
